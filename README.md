# hello-python

## Week 1 Summary

This repository contains my first Python learning project. It includes daily practice files from Week 1.

### What I learned in Week 1

- Day 1: Setup tools, configure GitHub SSH, and run a `hello.py` script.
- Day 2: Python variables and data types (`int`, `float`, `str`, `bool`).
- Day 3: Control flow with `if`, `elif`, and `else`.
- Day 4: Loops and lists with `for` loops, `range()`, and list operations.
- Day 5: Functions, parameters, return values, and reusable code.
- Day 6: Practice problems with dictionaries and nested data.
- Day 7: Write this README and commit the final Week 1 changes.

## Files in this repo

- `hello.py` - First Hello World script.
- `day2.py` - Variables, types, and basic expressions.
- `profile_card.py` - A profile card built with variables and output formatting.
- `day3.py` - Decision-making examples using `if` statements.
- `day4.py` - Lists and loops practice.
- `day5.py` - Functions practice with reusable code blocks.
- `day5_challenge.py` - Day 5 challenge exercises.
- `day6.py` - Dictionary practice and nested data.
- `README.md` - Week 1 summary and instructions.

## Run the code

Run any file using Python 3 in the project folder:

```bash
python3 day2.py
python3 day3.py
python3 day4.py
python3 day5.py
python3 day5_challenge.py
python3 day6.py
```
